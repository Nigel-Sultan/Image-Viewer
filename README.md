# Image-Viewer
Computer Graphics

•	R resets all values
•	E closes window
•	W scales image up
•	S scales image down
•	D rotates counter-clockwise
•	A rotates clockwise
•	Right Arrow scales X up
•	Left Arrow scales X down
•	Up Arrow scales Y up
•	Down Arrow scales Y down
•	H translates x up
•	F translates x down
•	T translates y up
•	G translates y down
•	L shears x up
•	J shears x down
•	I shears y up
•	K shears y down
